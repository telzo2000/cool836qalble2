# cool836qalble2

![](img/img00021.jpg)

cool836qalble2 is the successor to cool836qalble.
<br>
cool836qalble2 connects to your PC via Bluetooth.
<br>
cool836qalble2 is a keyboard with a joystick (5 position switch).
<br>
